## Overview  
The Smart Vacuum Cleaner Robot is an Arduino-based automatic cleaning robot designed to reduce manual effort.  
It can detect obstacles and move smoothly around the environment while performing efficient floor cleaning.

## Key Features  
- Automatic navigation  
- Obstacle detection and avoidance  
- Motor-driven cleaning mechanism  
- Embedded system automation  

## Technologies Used  
- Arduino UNO  
- Ultrasonic Sensor  
- Motor Driver Module (L298N)  
- DC Motors and Wheels  
- Embedded C Programming  

## Applications  
- Home and indoor floor cleaning  
- Automation-based robotics projects  
- Embedded system learning and development  

## Future Enhancements  
- IoT-based remote control  
- Smart mapping and path planning  
- Improved cleaning efficiency
